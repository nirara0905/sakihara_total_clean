## ページ構成

1ページのランディングページ（LP）として構成する。

### セクション順序

```
1. Header（ロゴ + ナビゲーション）
2. Hero（ファーストビュー）
3. Services（事業紹介）
4. Reasons（選ばれる3つの理由）
5. Works（施工事例）
6. Contact（LINE誘導）
7. Footer（会社概要）
```

---

## 各セクション詳細

### 1. Header
- ロゴ（左）
- ナビゲーションリンク（各セクションへのアンカーリンク）
- スマホ時はハンバーガーメニュー

### 2. Hero
- キャッチコピー：「より良いお住いのお手伝いをします」
- サブコピー：会社の信頼感を伝える一文
- CTAボタン：「LINEで無料相談」→ LINEリンクへ遷移
- 背景：3枚の施工写真がふんわり切り替わるスライドショー（CSS `@keyframes` のみ、JS・ライブラリ不使用）
- 画像仕様：`object-fit: cover` で中央基準トリミング
  - 推奨解像度：1920 × 1080px（16:9）
  - 推奨ファイルサイズ：500KB以下
  - 撮影注意点：人物・重要な被写体は画像中央（左右25%以内）に配置。端は空・地面など切れても問題ない余白にする

### 3. Services（事業紹介）
3つの事業をカード形式で表示。順番：ガーデニング → 建築 → 清掃

| 事業 | 内容 |
|------|------|
| ガーデニング事業 | 庭木の剪定・植栽、草刈・除草、剪採・筑根、雑草対策・管理作業 |
| 建築事業 | リフォーム工事（内装）、外構工事（ウッドデッキ・駐車場）、塗装工事（防水・遮熱・外壁） |
| 清掃事業 | ハウスクリーニング、定期清掃 |

### 4. Reasons（選ばれる3つの理由）
- **① 価格** — 全て自社施工だから、高い作業品質と低価格を実現
- **② 技術** — 業界10年以上の植木職人が施工
- **③ 管理** — お庭・植木を美しく保つ継続管理

### 5. Works（施工事例）
- 完成形（アフターのみ）の写真ギャラリー
- 各事業2枚ずつ、計6枚掲載

### 6. Contact（LINE誘導）
- フォームなし
- **LINEボタン**のみ設置（LINE公式アカウントへのリンク）
- 電話番号も併記

## 画像ファイル命名規則

### 基本ルール
```
{セクション}_{内容}_{連番}.{拡張子}
```

- 全て小文字・ハイフン区切り（URLに使われるため大文字・スペースNG）
- セクション名を先頭にする（ファイル一覧でグループが分かりやすい）
- `before` / `after` を明示（ビフォーアフターの対応関係を明確にする）
- 末尾に連番 `_01`（同カテゴリで複数枚あっても管理しやすい）
